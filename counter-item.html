Counter

javascript project

This project is called Counter. This project uses JavaScript to increase and decrease the number displayed.

When the number is increased it turns green and ascends, when the number is decreased it descends. If the decrease button is pressed when the zero is displayed then the number displayed will be a negative number and the number displayed will turn red.
